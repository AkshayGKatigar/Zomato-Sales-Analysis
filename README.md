## Project Overview

This project focuses on analyzing **Zomato restaurant data** to uncover meaningful insights about restaurant distribution, customer preferences, pricing trends, and service availability across different countries and cities.

## Problem Statement

Zomato operates across multiple countries with thousands of restaurants, cuisines, and cities.
However, without proper analysis, it is difficult to understand:

* Which countries and cities dominate the restaurant ecosystem
* Availability of services like **online delivery** and **table booking**
* Popular restaurant chains with diverse cuisines
* Price distribution across restaurants
* Trends in restaurant openings over time



## Solution Approach

* Cleaned and processed raw Zomato dataset
* Created KPIs and visualizations to track performance metrics
* Built an interactive dashboard for easy exploration and filtering
